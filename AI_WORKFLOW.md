# AI Tool Workflow

This project was developed using AI-powered development tools to accelerate development and ensure code quality.

## Development Tools

### Cursor IDE / VS Code with AI Extensions

Setup:
- Editor: VS Code or Cursor
- AI Extension: GitHub Copilot / Cursor AI
- Python extension enabled

Workflow:

1. Code Generation
   - Used AI to generate boilerplate code for FastAPI endpoints
   - AI-assisted Pydantic schema creation
   - Automated regex pattern generation

2. Code Review
   - AI-powered code suggestions
   - Automatic bug detection
   - Best practices recommendations

3. Documentation
   - AI-assisted docstring generation
   - Automated README creation
   - Comment generation

4. Debugging
   - AI-powered error explanation
   - Solution suggestions
   - Code optimization tips

FastAPI Setup:
"Create a FastAPI endpoint for clinical note extraction with Pydantic validation"

Metrics Calculation:
"Write a function to calculate precision, recall, and F1-score for medication extraction"

Streamlit Dashboard:
"Build a Streamlit dashboard with tabs for extraction, metrics, and about sections"

"Create regex patterns to extract medications with name, dosage, and frequency from clinical notes"

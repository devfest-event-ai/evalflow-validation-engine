# EvalFlow - Production AI Validation Framework

## 🎯 Goal
Production-ready validation engine for healthcare AI outputs. 
Designed for Dexter Health Applied AI Engineer application.

## 🛠 Tech Stack
- FastAPI (Backend)
- Pydantic (Validation)
- Groq (LLM Inference)
- Streamlit (Dashboard)
- Pytest (Testing)

## 🚀 Quick Start
1. `pip install -r requirements.txt`
2. `uvicorn main:app --reload`
3. `streamlit run app.py`
